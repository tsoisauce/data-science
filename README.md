## Installation
Install your virtual environment and package manager: `pipenv` 
```bash
pip install pipenv
```
Create your Python 3 environment `pipenv`
```bash
pipenv --python 3.7.3
``` 
Tunnel into your environment
```bash
pipenv shell
```
Install libraries
```bash
pipenv install
```
